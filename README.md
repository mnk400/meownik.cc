# meownik.cc

A simple website that displays a random cat image on each page load.

## Features

- Random cat image from [Cataas](https://cataas.com) API
- Click the cat or press spacebar to load a new one
- Fade-in animation on image load
- Graceful error handling with retry

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Deployed to Cloudflare Pages at [meownik.cc](https://meownik.cc).
